# Ziptrrip Todo Application

## Description
This is a multi-page Todo application built using React frontend and Node.js Express backend.

## Features
- Add todo
- View all todos
- View single todo details using query parameter
- Mark todo as completed or pending
- Delete todo
- JSON file storage

## Tech Stack
- React
- Vite
- Axios
- React Router DOM
- Node.js
- Express.js
- CORS

## Pages
- `/` - Todo list page
- `/todo?id=TODO_ID` - Single todo details page

## Backend APIs
- GET `/todos`
- GET `/todos/:id`
- POST `/todos`
- PUT `/todos/:id`
- DELETE `/todos/:id`

## How to Run Backend

```bash
cd backend

node server.js
