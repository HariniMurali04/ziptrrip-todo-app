# Features Documentation

## 1. Todo List Page

### Features

* Displays all todos.
* Add a new todo.
* Mark todo as completed.
* Delete a todo.
* Navigate to the Todo Details page.

### Screenshot

![Todo List](screenshots/todo-list.png)

---

## 2. Todo Details Page

### Features

* Displays complete information about a selected todo.
* Receives the todo ID using a query parameter.
* Fetches todo details from the backend.
* Displays:

  * Todo ID
  * Title
  * Description
  * Status (Completed/Pending)

### Screenshot

![Todo Details](screenshots/todo-details.png)

---

## 3. Backend API

### Features

* RESTful CRUD APIs built with Express.js.
* JSON file used as data storage.
* Supports:

  * Create Todo
  * Read All Todos
  * Read Single Todo
  * Update Todo
  * Delete Todo
